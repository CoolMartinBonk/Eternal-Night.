# Eternal Night
> A generative ambient night scene built with React, Canvas API, and Web Audio API.

![Project Preview](scene.png)

## Tech Stack
- **Core**: React 18, TypeScript
- **Build Tool**: Vite
- **Graphics**: HTML5 Canvas API (2D Context)
- **Audio**: Native Web Audio API
